Construcción de Software
Guía de navegación

Esta clase deberá contar con algunas funciones que permitan, como mínimo: imprimir la información del empleado en pantalla, asignar un nombre, un servicio/departamento, asignar una posición y un salario.
Igualmente deberá existir un método para leer de cada instancia las informaciones importantes (nombre, servicio/departamento, posición y salario).
El programa (código fuente y archivo compilado) deberá ejecutarse en continuo, a través de una simple interfaz por línea de comando, con la opción de una tecla de salida, en el momento de finalizar el programa (por ejemplo, presione la letra X para salir).
El código y el programa deberán ser 100% funcionales.

1.10 Elabore un informe técnico describiendo paso a paso las elecciones hechas para el diseño e implementación de su programa.

1.11 Los archivos de código fuente y el ejecutable deberán agregarse como soporte al informe técnico que debe ser entregado a través del aula virtual.
Para continuar con la navegación del ambiente virtual de aprendizaje, despliegue el menú general ubicado en la esquina superior izquierda e ingrese a cada guía.
